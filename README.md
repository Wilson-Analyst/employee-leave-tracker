# Employee Leave Tracker System

A full-stack web application that allows employees to submit leave requests and allows admins to manage employees and approve/reject leave applications.

## 1. Features

✔ Employee management (Add, View, Update, Delete)  
✔ Submit leave requests with reason and duration  
✔ Approve/Reject leave requests  
✔ Login authentication (basic)  
✔ Dashboard with statistics  
✔ Frontend (React) + Backend (Spring Boot + MySQL)

---

## 2. Tech Stack

| Layer        | Technology                     |
|--------------|---------------------------------|
| Backend      | Spring Boot (Java)             |
| Frontend     | React (JavaScript)             |
| Database     | MySQL                          |
| API Testing  | Postman                        |
| Versioning   | Git & GitHub                   |
| IDEs         | IntelliJ IDEA (Backend), VS Code (Frontend)
